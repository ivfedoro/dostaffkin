# Dostaffkin

Приложение для заказа и отслеживания курьерской доставки.

## Функционал

- **Заказ доставки**: Форма с маршрутом (откуда/куда), размером посылки, скоростью доставки, контактными данными.
- **Отслеживание**: Поиск статуса доставки по номеру заказа.
- **Карта**: Интеграция с Yandex Maps для расчета маршрута и стоимости.

## Технологии

- **Frontend**: Angular 21, TypeScript
- **UI**: Reactive Forms, ngx-toastr
- **API**: RxJS, HttpClient
- **Карты**: Yandex Maps API
- **Сборка**: Angular CLI, Vite
- **Тестирование**: Vitest, JSDOM

## Установка и запуск

```bash
npm install
npm start
```

Приложение запустится на `http://localhost:4200`.
